Timeflux AMTI driver
====================

This package contains the node and related DLL files to acquire from a
AMTI force platform using timeflux_.

Go directly to the :py:class:`timeflux_amti.nodes.driver.ForceDriver`
documentation for details on how to start using this node to acquire
real-time data from the AMTI force platform.
